# Tag

Tags can be used by users to select a specific ingredient or category from a list of tags.
Once it is selected, it will be added to the search field.

## Default tag

Users should be able to click or press on a specific tag.

<Preview>
  <Story name="default">
    <Tag
      tag={{
        id: "1",
        title: "Tomaten",
        state: TagStates.TAG_UNSELECTED
      }}
    />
  </Story>
</Preview>

## Selected tag

Users should be able to remove selected tags.

  <Story name="selected">
